== Description ==

Dismiss, <strong>Gutenberg nag</strong>, dashboard widget, when is activated, or automatically, when put on mu-plugins directory.

> **Plugin Development**
>
> [GitHub](https://github.com/luciano-croce/gutenberg-nag/) official plugin development page.
== Installation ==
* Supported Blog or CMS installations:
 * Single WordPress (as plugin feature)
 * Single WordPress (as mu-plugins directory)
 * Multisite WordPress (as plugin feature)
 * Multisite WordPress (as mu-plugins directory)
= Step by Step Installation =
= For users of single WordPress 4.8+ or later (via FTP) =
1. Download Gutenberg Nag from WordPress.org plugin repository.
1. Upload it into /wp-content/plugins`/gutenberg-nag/` via FTP.
1. Activate Gutenberg Nag.
= For users of single WordPress 4.8+ or later (manual) =
1. Download Gutenberg Nag from WordPress.org plugin repository.
1. Upload it into your WordPress directly from Plugin Add Feature.
1. It will create a directory /wp-content/plugins`/gutenberg-nag/`
1. Activate Gutenberg Nag.
= For users of single WordPress 4.8+ or later (auto) =
1. Search Gutenberg Nag from Plugin Add Feature.
1. Install it live directly from WordPress.org repository.
1. It will create a directory /wp-content/plugins`/gutenberg-nag/`
1. Activate Gutenberg Nag.
= For users of multisite WordPress 4.8+ or later (via FTP) =
1. Download Gutenberg Nag from WordPress.org plugin repository.
1. Upload it into /wp-content/plugins`/gutenberg-nag/` via FTP.
1. Activate Gutenberg Nag.
= For users of multisite WordPress 4.8+ or later (manual) =
1. Download Gutenberg Nag from WordPress.org plugin repository.
1. Upload it into your WordPress directly from Plugin Add Feature.
1. It will create a directory /wp-content/plugins`/gutenberg-nag/`
1. Activate Gutenberg Nag.
= For users of multisite WordPress 4.8+ or later (auto) =
1. Search Gutenberg Nag from Plugin Add Feature.
1. Install it live directly from WordPress.org repository.
1. It will create a directory /wp-content/plugins`/gutenberg-nag/`
1. Activate Gutenberg Nag.
= For users of single WordPress mu-plugins directory =
1. Put this single file gutenberg-nag.php (not its parent directory) in the /wp-content/mu-plugins/ directory (create it if not exists).
= For users of multisite WordPress mu-plugins directory =
1. Put this single file gutenberg-nag.php (not its parent directory) in the /wp-content/mu-plugins/ directory (create it if not exists).
= How to uninstall Gutenberg Nag =
1. Disable Gutenberg Nag from Menu Plugins of Control Panel.
1. Delete Gutenberg Nag from Menu Plugins of Control Panel.
= Troubleshooting =
If all else fails and your site is broken remove directly via ftp on your host space /home/your-wp-install-dir/wp-content/plugins`/gutenberg-nag/` or /home/your-wp-install-dir/wp-content/mu-plugins`/gutenberg-nag.php/`.
= WordPress Multisite =
Remenber That: on WordPress MultiSite Gutenberg Nag Plugin, is available only on Super Admin Menu.
== Frequently Asked Questions ==
`
Tips: a neat trick is to put this single file gutenberg-nag.php (not its parent directory) in the /wp-content/mu-plugins/ directory (create it if not exists) so you won't even have to enable it, and will be loaded by default, also, since first step of installation!
`
